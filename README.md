# Mastermind Number Guessing Game (C Program)

This project is a console-based **Mastermind-style number guessing game** written in C.  
The user must guess a secret number, and after each attempt the program provides helpful hints such as:

- Correct digit in the **correct position**
- Correct digit in the **wrong position**

The game continues until the user successfully guesses the secret number.
